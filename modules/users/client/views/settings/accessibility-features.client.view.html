OpenDyslexic is distributed under a modified MIT license. This means it is free and easy to include in Pronouny.

OpenDyslexic will start working instantly. If you are logged in on other devices, you will need to reload the app. If you need to disable it, you will need to refresh the app.

"OpenDyslexic is created to help with some of the symptoms of dyslexia. Letters have heavy weighted bottoms to indicate direction. You are able to quickly figure out which part of the letter is down which aids in recognizing the correct letter, and sometimes helps to keep your brain from rotating them around. Consistently weighted bottoms can also help reinforce the line of text. The unique shapes of each letter can help prevent confusion through flipping and swapping."

Increasing text size may cause interface errors and style glitches.

You will need to reload the page to see the font size change. Pronouny defaults to 14px and maintains style to about 22px.

Pronouny allows you to change the base font size which will affect all text expect headings.

Higher contrast is not yet available. When it becomes available, you will automatically be switched over.

You may have to clear your cache and refresh to enter higher contrast mode.

Pronouny attempts to use acceptable contrast colours accross the website by default. Pages are regularly checked for text contrast and tweaked as needed.
